# Portfólio Pessoal - Guilherme Machado

Este é o meu portfólio pessoal desenvolvido com o objetivo de apresentar meus principais projetos, habilidades e experiências na área de desenvolvimento web front-end.

O site reúne projetos práticos criados com HTML, CSS, JavaScript e Bootstrap, demonstrando minha evolução como desenvolvedor, minha capacidade de estruturar interfaces modernas e meu foco em criar páginas responsivas, organizadas e funcionais.

## Sobre o projeto

O portfólio foi desenvolvido para servir como uma vitrine profissional, reunindo informações sobre minha trajetória, tecnologias que utilizo, projetos desenvolvidos e formas de contato.

A proposta é apresentar, de maneira clara e objetiva, minhas habilidades em desenvolvimento front-end, além de destacar projetos que simulam situações reais do mercado, como sites institucionais, sistemas de login, cardápios digitais, aplicações de clima e listas de tarefas.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Font Awesome
- Git
- GitHub Pages

## Funcionalidades

- Página inicial com apresentação profissional
- Seção sobre mim
- Área de habilidades técnicas
- Cards de projetos com links para visualização
- Layout responsivo para diferentes tamanhos de tela
- Navegação simples e intuitiva
- Área de contato
- Design moderno e organizado

## Projetos em destaque

### Site Empresarial - Construção, Reforma e Manutenção

Site institucional desenvolvido para uma empresa fictícia/prestadora de serviços na área de construção, reformas e manutenção residencial/comercial.

---

### Cardápio Digital / Delivery

Projeto de cardápio digital voltado para restaurantes, lanchonetes e serviços de delivery, com apresentação de produtos, preços e categorias.

---

### Página de Login e Cadastro

Interface moderna de login e cadastro, desenvolvida com foco em design visual, organização dos campos e experiência do usuário.

---

### Clima App

Aplicação web que permite consultar informações climáticas de uma cidade, exibindo dados como temperatura, condição do tempo e localização.

---

### Lista de Tarefas Avançada

Aplicação de lista de tarefas com funcionalidades para adicionar, marcar, editar e remover atividades, auxiliando na organização do dia a dia.


## Objetivo

O objetivo deste portfólio é demonstrar minha evolução prática como desenvolvedor front-end, apresentando projetos reais e funcionais que evidenciam meus conhecimentos em estruturação de páginas, estilização, responsividade, interatividade e organização de código.

## Aprendizados

Durante o desenvolvimento deste portfólio e dos projetos apresentados, aprimorei conhecimentos importantes em:

- Estruturação semântica com HTML
- Estilização moderna com CSS
- Criação de layouts responsivos
- Manipulação do DOM com JavaScript
- Organização de arquivos e componentes
- Publicação de projetos no GitHub Pages
- Boas práticas de versionamento com Git
