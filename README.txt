PORTFÓLIO DEV GuiCRM - HTML, CSS e JavaScript

Arquivos principais:
- index.html
- style.css
- script.js
- assets/

Como usar:
1. Abra o arquivo index.html no navegador.
2. Para trocar a foto, coloque sua imagem em assets/foto-perfil.jpg.
3. Para trocar a imagem principal, coloque uma imagem em assets/banner-tech.jpg.
4. Para ativar o botão Baixar CV, coloque seu currículo em assets/curriculo-guilherme-machado.pdf.
5. Edite links de GitHub, LinkedIn, WhatsApp e projetos diretamente no index.html.

Para subir no GitHub Pages:
1. Crie um repositório.
2. Envie index.html, style.css, script.js e a pasta assets.
3. Vá em Settings > Pages > Deploy from branch > main > /root.
